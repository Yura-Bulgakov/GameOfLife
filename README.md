# Клеточный автомат «Game Of Life»

## Пререквизиты

Для запуска приложения необходимо:
* jdk11;
* git;
* make;
* curl.

> Пути к данным утилитам должны быть указаны в переменной $PATH.

## Запуск

Windows

    git clone https://github.com/Yura-Bulgakov/GameOfLife.git
    cd GameOfLife
    make wpackage
    java -jar app.jar

Linux/MacOS

    git clone https://github.com/Yura-Bulgakov/GameOfLife.git
    cd GameOfLife
    make package
    java -jar app.jar
